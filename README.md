Web Portal Repository (Insighta Web)
The Web Portal provides a user-facing dashboard for interacting with profile data in a browser environment. It is developed using React and deployed on Vercel to ensure seamless integration with the backend API. When a user authenticates via the web, the portal receives an access token stored in a secure, HTTP-only cookie to prevent cross-site scripting attacks. The dashboard allows users to view their GitHub profile information, including their username and avatar, which is retrieved from the backend after a successful OAuth handshake. It also manages the frontend state to handle session expirations in accordance with the short-lived token requirements established in the system architecture.
